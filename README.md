## 🎨 Design Decisions

### Color Palette
- **Primary Red:** #F53838 (buttons, accents)
- **Dark Text:** #0B132A (headings)
- **Gray Text:** #4F5665 (body text)
- **Light Background:** #F8F8F8 (sections)
- **White:** #FFFFFF (cards, background)

### Typography
- **Font Family:** Rubik (Google Fonts)
- **Weights:** 300, 400, 500, 600, 700
- **Hierarchy:** Clear heading sizes for visual flow

### Layout Approach
- **Mobile-first CSS** with media queries
- **Flexbox** for navigation and simple layouts
- **CSS Grid** for pricing cards and sponsors
- **Container class** for consistent max-width

## 🚧 Challenges & Solutions

### Challenge 1: Mobile Navigation
**Problem:** Navigation links took too much space on mobile screens  
**Solution:** Implemented checkbox-based hamburger menu without JavaScript  
**Result:** Clean mobile interface with toggleable menu

### Challenge 2: Responsive Pricing Cards
**Problem:** 3 cards needed to stack properly on mobile but display side-by-side on desktop  
**Solution:** Used CSS Grid with `grid-template-columns: repeat(auto-fit, minmax(280px, 1fr))`  
**Result:** Cards automatically adjust based on screen width

### Challenge 3: SVG Image Scaling
**Problem:** SVG illustrations weren't scaling properly on different screen sizes  
**Solution:** Set `max-width: 100%` and `height: auto` on all images  
**Result:** Images scale smoothly without distortion

### Challenge 4: Testimonials Layout
**Problem:** Testimonials needed to be visually appealing without JavaScript slider  
**Solution:** Created card grid that wraps naturally; arrows are visual placeholders  
**Result:** Clean presentation that works without complex functionality
